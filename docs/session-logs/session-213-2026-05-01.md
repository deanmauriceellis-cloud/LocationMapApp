## Phase 1 вЂ” `normalizeForTts(text)` in NarrationManager

New `internal fun normalizeForTts(input: String): String` runs once per segment in `speak()` (line 530 area) before `chunkOnPunctuation`. Ten-stage pipeline:

1. Strip Markdown emphasis: `**bold**`, `__bold__`, `*italic*`, `_italic_`, `~~strike~~`. Negative-lookbehind/lookahead on alpha so single asterisks in arithmetic ("3 * 4 = 12") survive.
2. **Titles** (`Capt./Rev./Mr./Mrs./Dr./Sgt./Maj./Col./Hon./Prof./Dea./Gov./Sen./Lt./Adm./...` 23 entries) вЂ” drop period unconditionally before any alpha word. Titles always precede a name; period is never sentence-terminal.
3. **St.** вЂ” explicit saint-name allowlist (~50 names: Joseph, Peter, Paul, John, Mary, Anne, Nicholas, Edmunds, ...) в†’ "Saint". Otherwise в†’ "Street" (period kept on `\s+[A-Z]|\s*$` for sentence-end safety, dropped mid-sentence). Resolves the `Orne St. Events` в†’ `Saint Events` ambiguity.
4. **States** (`Mass./Conn./Calif./Fla./Penn./Vt.`) вЂ” keep period when sentence-terminal, drop mid-sentence.
5. **Trailing street suffixes** (`Ave./Blvd./Rd./Ln.`) вЂ” same heuristic as states.
6. **Leading place prefixes** (`Mt./Ft.`) вЂ” drop period before alpha (behave like titles, e.g. `Mt. Vernon` в†’ `Mount Vernon`).
7. **Postnominals** (`Jr./Sr.`) вЂ” sentence-starter allowlist (`He|She|It|They|This|After|Before|...`) keeps period; otherwise drop. Disambiguates `Joseph Jr. House` (mid-sentence) vs `Joseph Jr. He died` (sentence end). `Esq.` uses simpler `\s+[A-Z]|\s*$`.
8. **Company suffixes** (`Bros./Inc./Co./Ltd./Corp.`) вЂ” same heuristic as states.
9. **Mid-sentence connectives** (`ca./etc./vs./approx.`) вЂ” drop period, expand to full word.
10. **Era markers** (`B.C./A.D.`) вЂ” keep period if sentence-terminal.

Final pass collapses double spaces.
