### Subtopic auto-gen pipeline

New script `cache-proxy/scripts/auto-gen-narration-subtopics.js`. Synthesizes `narration_subtopics` candidates from local sources only (provenance-pause-respecting):

1. **Figure detection** — scans each POI's combined narration text (`long || short || historical || description || name`) for any of the 49 active NPC names from `salem_witch_trials_npc_bios`. Multi-variant matching: dual-named NPCs ("Ann Carr Putnam — Ann Putnam Sr.") are split on em-dash, both halves searched. Variants <8 chars dropped to prevent false hits on bare first names. Whole-word case-insensitive regex. Cap 3 figure cards per POI.
2. **Adjacency** — finds the single nearest active POI within 200 m that passes the eligibility fence and has a `long_narration` of ≥200 chars. Cap 1 adjacent card per POI.
3. **Eligibility fence** — operator rule, refined mid-session: only `category IN (HISTORICAL_LANDMARKS, HISTORICAL_BUILDINGS, CIVIC, WORSHIP)` OR `is_historical_property=true` OR `is_civic_poi=true` OR `is_witch_trial_site=true`. Commercial categories (Shopping, Food & Drink, Healthcare, Lodging, Education, etc.) are skipped entirely as both source and target. `is_tour_poi` is intentionally **NOT** included due to the data-quality pollution captured above.
4. **Dupe suppression** — `looksLikeDupe(name1, name2)` skips any adjacency target whose name overlaps the source POI's name by ≥50% of significant tokens or is a substring. Suppresses cross-links between known dupes (`yin_yu_tang ↔ yin_yu_tang_house_peabody_essex_museum`, `witch_trials_memorial ↔ salem_witch_trials_memorial`).
5. **Source-kind tagging** — every emitted card carries `source_kind: 'figure'` or `source_kind: 'adjacent_poi'` plus `source_ref: <id>`, so the admin SubtopicEditor (which renders `source_kind` in the per-card row) tells the operator which cards are auto-gen vs hand-authored.
6. **Default-skip on POIs with existing subtopics** — `--overwrite` flag forces re-run.
7. **Other flags** — `--dry-run`, `--limit N`, `--poi-id ID` for targeted probing.
