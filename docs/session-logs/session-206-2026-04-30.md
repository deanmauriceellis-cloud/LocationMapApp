## Phase 2 — Apply-categorization 413 fix

Real fix: bumped global limit `16kb → 1mb` in `server.js:25` (now matches the urlencoded limit). Verified with synthetic 211-row payload (~19 kB) → `{"applied":0,"requested":211}` with fake IDs.

## Phase 7 — Tier filter dropping tour-flagged POIs

Operator: "I am still not clear why Samantha's statue wasn't a shown POI when I was in tour mode?"

Walked the marker filter chain in `SalemMainActivity.loadNarrationPointMarkers` (lines 2208–2272):

```
loadNarrationPoints()  →  tierFiltered (2212-2218)  →  histModeFiltered (2223-2227)  →  layer gate (2249-2271)  →  marker
                            ❌ doesn't honor isTourPoi      ✅ pass-through              ✅ HONORS isTourPoi
                                                            (histMode=false this drive)
```

`tierFiltered` admits only `adPriority>0 OR HISTORICAL_BUILDINGS OR isHistoricalProperty OR isCivicPoi`. **`is_tour_poi=true` is NOT in the admission criteria** — so a tour-flagged PARKS_REC / EDUCATION / WORSHIP / ENTERTAINMENT / SHOPPING POI never reaches the layer gate where the override would fire.

**Numbers:**
- 222 `is_tour_poi=true` POIs in narrated pool
- **94 pass tier filter**
- **128 dropped** — by category: PARKS_REC 67, EDUCATION 36, WORSHIP 19, CIVIC 4 (already covered), ENTERTAINMENT 3, SHOPPING 2, WITCH_SHOP 1

Marker counts in operator's log match exactly: 94 / 386 / 449 / 157 across the four `(allowHist, allowCivic)` permutations.

**Samantha specifically** — `HISTORICAL_BUILDINGS` so she passes tierFiltered, reached layer gate, isTourPoi override admitted her. Marker WAS rendered. But **Lappin Park** (the park she stands in, `is_civic_poi=true, is_tour_poi=false`) disappeared at `allowCivic=false` — and without the park as a landmark, Samantha looked like just another HB dot in the cluster.

**Fix (one line):** added `p.isTourPoi` to the tier-filter admission criteria in `SalemMainActivity.kt:2212`:

```kotlin
allPoints.filter { p ->
    p.adPriority > 0 ||
    p.category == "HISTORICAL_BUILDINGS" ||
    p.isHistoricalProperty ||
    p.isCivicPoi ||
    p.isTourPoi   // ← S206: honor force-visible override at tier filter too
}
```
