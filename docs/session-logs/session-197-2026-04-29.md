## Witch-Trials Speak bypasses AudioControl gate (NarrationManager.kt + TourViewModel.kt)

- Root cause: `WitchTrialsMenuDialog` uses tags `witchtrials_article` / `witchtrials_newspaper` / `witchtrials_bio`. `NarrationManager.inferKindFromTag()` routes any `witchtrials_*` to `NarrationKind.ORACLE`.
