### Pre-work discovery

- Admin tool stack: React 19 + Vite on port 4302, proxies `/api/*` to cache-proxy at `http://10.0.0.229:4300` (Basic Auth via browser's native credential dialog on `/api/admin/*`).

- Edit dialog (`web/src/admin/PoiEditDialog.tsx`, 1,661 lines) uses Headless UI `Tab`/`TabPanel` with 7 tabs pre-S160: General / Location / Hours & Contact / Narration / Provenance / Linked Historical / Danger Zone.
- Field whitelist lives in two mirrored files: `cache-proxy/lib/admin-pois.js` `UPDATABLE_FIELDS` (62 fields pre-S160) and `web/src/admin/poiAdminFields.ts` `UPDATABLE_FIELDS` (same 62, mirrored intentionally per the file's own header). PG reads via `SELECT * FROM salem_pois` so the 9 new columns are already fetched — only the UPDATE whitelist and UI form needed work.
- Witchy tile archive: `tools/tile-bake/dist/salem_tiles.sqlite`, schema `tiles(key INTEGER, provider TEXT, tile BLOB, PRIMARY KEY (key, provider))`, key = osmdroid encoding `(BigInt(z) << BigInt(z + z)) | (BigInt(x) << BigInt(z)) | BigInt(y)`, three providers: `Salem-Custom` (WebP), `Mapnik` (PNG), `Esri-WorldImagery` (JPEG).

### Backend changes

1. **`cache-proxy/lib/admin-pois.js`** — appended 9 column names to `UPDATABLE_FIELDS` under a new "Phase 9Y — MassGIS / MHC Inventory / L3 parcel enrichment" block. Total count went 62 → 71.

2. **`cache-proxy/lib/admin-tiles.js` (new)** — 120-line module that opens `tools/tile-bake/dist/salem_tiles.sqlite` read-only (lazy, on first request), defines `osmdroidKey(z, x, y)` with `BigInt`, exposes `GET /admin/tiles/:provider/:z/:x/:y`. Validates provider against an allowlist of 3, strips optional file extension from `y` (Leaflet sometimes appends `.png`), rejects bad z/x/y, 404s on missing tile, 503 if the archive file is absent. Sniffs Content-Type from magic bytes — PNG `89 50 4E 47`, JPEG `FF D8 FF`, WebP `52 49 46 46 ?? ?? ?? ?? 57 45 42 50` — returns the blob with `Cache-Control: public, max-age=86400, immutable`.

3. **`cache-proxy/server.js`** — registered the new router at the same mount point as the other `/admin/*` modules so Basic Auth covers it.

### Frontend changes

4. **`web/src/admin/poiAdminFields.ts`** — mirrored the 9 field additions in `UPDATABLE_FIELDS`; added `mhc_year_built` to `NUMERIC_FIELDS` so the form renders it as `<input type="number">` with `step="1"`.

Added a `TileProviderPicker` component — top-right 1000-z-index floating box with a `<select>` listing all four labels.
