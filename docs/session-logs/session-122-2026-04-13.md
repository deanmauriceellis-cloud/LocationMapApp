### — AudioCraft installation for sound effect generation

**Goal:** Install AudioCraft (Meta's AudioGen) locally on RTX 3090 for generating custom sound effects for the Salem tour app — atmospheric sounds, geofence triggers, character effects.

**Installation path:** `~/AI-Studio/audiocraft/` (repo) + `~/AI-Studio/audiocraft-venv/` (Python 3.12 venv)

**Issues resolved during install:**
- PyPI audiocraft 1.3.0 pinned stale deps (spacy 3.5.2/thinc incompatible with Python 3.12) — installed from GitHub instead (1.4.0a2)
- GitHub version pinned torch==2.1.0 — relaxed requirements.txt, installed torch 2.11.0+cu130 via xformers dependency chain
- Missing deps (spacy, torchdiffeq, torchmetrics, demucs) — installed incrementally as AudioCraft's eager imports demanded them
- Model download stalled (incomplete 63.9MB blob) — cleared and re-downloaded full 3.5GB state_dict.bin
- `libnvrtc-builtins.so.13.0` not found — torch 2.11 ships CUDA 13.0 but system has CUDA 12.0. Fixed via LD_LIBRARY_PATH pointing to pip-installed nvidia libs
- torchaudio 2.11 save backend broken (torchcodec/libnppicc.so.13 missing) — switched to soundfile for WAV output

**Final stack:** AudioCraft 1.4.0a2, PyTorch 2.11.0+cu130, xformers 0.0.35, torchaudio 2.11.0, CUDA on RTX 3090 (24GB)

**Smoke test:** 3 clips generated successfully (5s each, 16kHz mono WAV):
1. "witch cackling while flying on a broomstick through the night sky"
2. "old creaking wooden door opening slowly in a haunted house"
3. "footsteps on cobblestone street at night"
