## Research: Frida GPS Walking Simulation

Researched approaches for GPS simulation on the rooted tablet:
- **Frida hook approach:** Hook FusedLocationProviderClient + LocationManager at Java layer, inject synthetic Location objects with `isFromMockProvider(false)`. Full control over all fields (lat, lng, bearing, speed, accuracy, altitude). Bypasses all mock detection.
- **Walking simulation:** Interpolate between waypoints at 1.2 m/s with ±2m GPS jitter, proper bearing calculation, realistic accuracy variation (3-8m).
- **Python control:** Start/pause/speed (5x/10x)/jump to stop N — leverages gog-agent's existing Frida bridge patterns.
- **Alternative (ADB mock provider):** Simpler but sets `isFromMockProvider(true)` — not suitable for production-like testing.

**Decision:** Custom Frida script is the right approach. Gives full control, bypasses mock detection, and the gog-agent infrastructure provides proven patterns.
