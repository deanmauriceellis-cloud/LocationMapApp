Minimal four-field create form (Name *, Category *, Subcategory optional, Address optional) plus read-only lat/lng captured from the map click. Mirrors the inline "+ Add new category" panel from PoiEditDialog so the operator can create categories on the fly during the create flow (operator: "I will create some new categories that we will refine later"). Uses Headless UI Dialog at z=1100 (one above the edit dialog at z=1000) with plain useState for the four fields — no react-hook-form, since it's a 4-field form not a 60-field one.

Cursor goes crosshair, an emerald banner ("Click the map to drop a new POI · Esc to cancel") shows at the top, and `ParcelHitTest` is suppressed in addPoiMode so a building polygon doesn't intercept the place click. Esc handler extended to cancel either mode.
