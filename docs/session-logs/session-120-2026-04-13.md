4. **PoiActionSynthesizer.kt** — `buildActions(poi: SalemPoi)`.
5. **PoiHeroResolver.kt** — Completely rewritten. Resolution order: (1) `poi.imageAsset` hero image from S119, (2) category-based fallback, (3) red placeholder. Supports S119's 1,013 hero images.
6. **ProximityDock.kt** — `onPoiTapped: ((SalemPoi) -> Unit)`, category colors and folder mappings updated to uppercase.
7. **TourViewModel.kt** — `loadNarrationPoints()` now calls `repository.getNarratedPois()` (queries `salem_pois WHERE is_narrated=1`).
8. **PoiDetailSheet.kt** — JSON serialization updated to SalemPoi fields. `poi.type` → `poi.category` with display formatting. `imageAsset` and `geofenceRadiusM` now serialized.
9. **SalemMainActivityNarration.kt** — Queue, current narration, all function signatures migrated. `voiceOverride` (absent on SalemPoi) replaced with direct CategoryVoiceMap lookup.
10. **SalemMainActivity.kt** — `narrationMarkers` type updated. Marker icon uses `p.category.lowercase()`. Snippet shows formatted category.
11. **MarkerIconHelper.kt** — Added SalemPoi unified category entries (food_drink, tourism_history, etc.) to CATEGORY_MAP for marker coloring.
