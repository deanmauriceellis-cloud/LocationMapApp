### Architecture decisions
- **D1** Dedicated `ActivationHttpClient` (no `OfflineModeInterceptor`, no `CertificatePinner`; `ActivationHostGuard` exact-host/no-redirect/no-cleartext) — preserves default-deny for the 14 gated clients.
- **D2** Gate in a new `ActivationActivity` between Splash and Main; activated launches verify the local receipt only (no network).
- **D3** Revalidate on VIOLATION, never on a timer (`ACTIVATED ⇄ LOCKED`).
- **D4** Tiered hashing — manifest sig + `salem_content.db` every launch; 354MB tile DB + image dirs are `unverifiedLarge` (size only, full-verify at activation).
- **D5** Debug bypass via `BuildDefaults.ACTIVATION_BYPASS = BuildConfig.RECON_DEFAULTS` (R8-strips in release).
- **D6** Stateless Worker, LMA-owned, `*.workers.dev`; HMAC time-boxed nonce (no KV); `decodeIntegrityToken` (SA-JWT via Web Crypto) reads licensing/recognition/device verdicts; retain-nothing; `EXPECTED_MANIFEST_HASHES` (current+previous) baked at deploy.
