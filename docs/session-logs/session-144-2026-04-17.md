### 2026-04-17 ‚Äî #50: TTS pause-at-sentence-end

- New `splitIntoSentences(text)` ‚Äî character walker that consumes terminator runs (`...`, `!?`, `!!`), then checks last-word-before-terminator against the abbreviation set, checks single-letter initial, and checks if the next non-whitespace char is lowercase (continuation). Only splits when all three say "real sentence end."
- `chunkForTts` now uses `splitIntoSentences` instead of the naive regex. Short text (< 3500 chars, Android TTS cap) still speaks in one utterance with natural Android prosody.
